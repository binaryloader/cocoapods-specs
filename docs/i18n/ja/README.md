# cocoapods-specs

[English](../../../README.md) | [한국어](../ko/README.md) | **日本語**

iOSユーティリティモジュール向けのプライベートCocoaPodsスペックリポジトリです。

## Specs

| Pod | 最新バージョン | 説明 |
|-----|--------------|------|
| BinaryLoaderUI | 1.0.2 | UIコンポーネント(InsetTextFieldなど) |
| BinaryLoaderNetwork | 1.0.4 | ネットワーキングレイヤー |
| BinaryLoaderExtensions | 1.0.2 | Swiftエクステンション |
| BinaryLoaderDIContainer | 1.0.4 | 依存性注入コンテナ |

## 使い方

`Podfile`にこのスペックリポジトリをソースとして追加します。

```ruby
source 'https://github.com/binaryloader/cocoapods-specs.git'
```

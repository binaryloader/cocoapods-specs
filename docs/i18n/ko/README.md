# cocoapods-specs

[English](../../../README.md) | **한국어** | [日本語](../ja/README.md)

iOS 유틸리티 모듈을 위한 비공개 CocoaPods 스펙 저장소이다.

## Specs

| Pod | 최신 버전 | 설명 |
|-----|----------|------|
| BinaryLoaderUI | 1.0.2 | UI 컴포넌트(InsetTextField 등) |
| BinaryLoaderNetwork | 1.0.4 | 네트워킹 레이어 |
| BinaryLoaderExtensions | 1.0.2 | Swift 익스텐션 |
| BinaryLoaderDIContainer | 1.0.4 | 의존성 주입 컨테이너 |

## 사용 방법

`Podfile`에 이 스펙 저장소를 소스로 추가한다.

```ruby
source 'https://github.com/binaryloader/cocoapods-specs.git'
```
